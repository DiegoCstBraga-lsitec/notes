# Apontamentos

## Rodar servidor local

1. Abra o terminal
2. Insira o seguinte comando 
```bash
psql -U postgres
```
3. Insira a senha do servidor
4. Agora com o servidor postgres ligado, na pasta raiz do projeto, insira o comando 
```bash
node server/src/server.js
```
